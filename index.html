Connect to Google Sheets

Enter your details and paste the Google Apps Script Web App URL to connect this dashboard to the live cheque register. You only do this once per device.

1

Create Google Sheet + paste Apps Script

Go to your ISMA Cheque Register Google Sheet → Extensions → Apps Script → paste ISMA_AppScript.gs → Save

2

Run setupSheets once

Select setupSheets from dropdown → Run. Creates MAIN 102, QAB 107 and Planned Expenses tabs.

3

Deploy as Web App

Deploy → New Deployment → Web App → Execute as: Me → Who has access: Anyone → Deploy → copy URL below

https://script.google.com/macros/s/YOUR_ID/exec
